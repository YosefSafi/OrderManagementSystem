# Order Management System (Microservices)

This project is a microservices-based order management system built with C# and .NET 9. It demonstrates inter-service communication using synchronous (REST) and asynchronous (Message Broker) patterns.

## Architecture Highlights
* **Order API**: Core order management.
* **Inventory API**: Stock tracking and reservations.
* **Payment API**: Transaction processing.
* **Notification API**: Asynchronous alerts.
* **API Gateway (YARP)**: Unified entry point.
* **Message Broker (RabbitMQ)**: Event-driven communication.
